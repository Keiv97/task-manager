# Task Manager 📝

Eine einfache Web-App zur Aufgabenverwaltung – gebaut mit Python, Flask und SQLite.

## Was kann die App?

- Aufgaben hinzufügen
- Aufgaben als erledigt markieren
- Aufgaben löschen
- Daten werden dauerhaft in einer SQLite-Datenbank gespeichert

## Technologien

- **Python** – Programmiersprache
- **Flask** – Web-Framework für den Server
- **SQLite** – Datenbank
- **HTML/CSS** – Frontend

## Lokale Installation

1. Repository klonen:
```bash
   git clone https://github.com/Keiv97/task-manager.git
   cd task-manager
```

2. Virtuelle Umgebung erstellen und aktivieren:
```bash
   python3 -m venv venv
   source venv/bin/activate
```

3. Flask installieren:
```bash
   pip install flask
```

4. App starten:
```bash
   python3 app.py
```

5. Browser öffnen: `http://localhost:5000`
